# Stocker
## 개요
Stocker는 대지고등학교 동창들이 만드는 주식정보 Discord 봇입니다!

## 사용법 | How to Use
### 1. 개인 가상환경 구축
```
python -m venv venv
```
```
venv\Scripts\activate
```

### 2. 필요 패키지 설치
가상환경이 구축된 cmd 위에서 실행해야 한다. 가상환경이면 터미널에 (venv)가 보인다.
```
pip install -r requirements.txt
```
